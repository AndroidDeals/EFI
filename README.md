# i5-10500es-Opencore0.6.0-Hackintosh
按照YouTube司波图大佬的装机配置采购的硬件。折腾了之后成功吃上了Big Sur。

硬件列表
1. 处理器 i5-10500ES
2. 主板 ~~昂达B460SD4~~ 
   现换成微星B460M迫击炮😁，因为昂达的bios太复杂，设置BIOS之后它自杀了，直接黑屏。
3. 独显 映泰RX470
4. 内存 阿斯加德8g两个
5. 固态 海力士啥型号256g
